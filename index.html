HTML y CSS para principiantes.

HTML Y CSS son dos herramientras importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este curso aprenderás todo sobre HTML y CSS, desde lo mas básico hasta un nuvel intermedio, logrando desarrollar tu priemra página web responsiva, sun la necesidad de aplicar librerías externas para CSS.

Instructor: Manuel González
Estilos con SASS.

SASS es una herramienta importante hoy en día para todo desarrollador web, ya que permite crear estilos mas odenados, eficientes y con menos líneas de código. Por ello, en este curso aprenderás todo lo necesario para crear hojas en estilos desde la sintaxis de SCSS, implementado técnicas propias de SASS.

Instructor:Manuel González
